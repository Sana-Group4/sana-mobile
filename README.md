## Run locally

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app (using expo)

   ```bash
   npx expo start
   ```

3. Make sure you have the sana backend running on the same network, it is available at the following repository [here](https://github.com/Sana-Group4/sana-backend). The address for the backend must be set in the .env file (see sample .env for correct variable naming) in the sana-mobile repo. If running the app with expo android healthkit integration will not be available.
